# 伏羲 (Fuxi) - 初中语文学习系统

## 项目概述

这是一个初中语文教育学习项目，目前包含一个完整的初一语文上册核心知识点闯关应用。该项目旨在通过互动式答题帮助学生掌握语文教材中的核心知识。

### 主要特性

- **300道核心知识点**：覆盖人教版七年级上册语文全部六个单元
- **自适应学习算法**：答对的题目出现频率降低，连续三次答对视为"已掌握"
- **进度持久化**：使用 LocalStorage 保存学习进度
- **游戏化体验**：连对提示、进度条、正确率统计
- **响应式设计**：支持移动端和桌面端

## 题型分类

- **文学常识**：作者、时代、作品出处
- **字音**：易错读音辨析
- **字形**：易错字形辨析
- **词语解释**：重点词语含义
- **默写**：古诗文填空
- **课文理解**：文章主旨、结构分析
- **修辞手法**：比喻、拟人、排比等
- **字词解释**：文言文实词虚词
- **句子翻译**：文言文翻译

## 使用方法

1. 直接在浏览器中打开 HTML 文件即可使用
2. 答题进度会自动保存到浏览器本地存储
3. 连续三次答对同一题目后，该题目将被标记为"已掌握"
4. 可通过"重新开始"按钮重置所有进度

## 技术实现

### 数据结构

```javascript
// 知识点对象示例
{
  id: 1,
  unit: "第一单元",
  lesson: "《春》",
  type: "文学常识",
  question: "《春》的作者朱自清是哪个时期的著名散文家？",
  options: ["现代", "当代", "近代", "古代"],
  answer: 0,
  explanation: "朱自清（1898-1948）是中国现代著名散文家、诗人、学者。"
}

// 学习状态
{
  frequency: 10,        // 出现频率权重
  consecutiveCorrect: 0 // 连续正确次数
}
```

### 自适应算法

- 初始每道题权重为 10
- 答对一次：权重减 2（最低为 1）
- 答错一次：权重重置为 10
- 连续正确 3 次：标记为已掌握，不再出现
